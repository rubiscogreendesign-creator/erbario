Token GitHub personale

Per aggiungere schede direttamente al repository senza copia-incolla, serve un "fine-grained personal access token" GitHub, limitato al solo repo erbario.

Come generarlo (una volta sola):

  1. Apri github.com/settings/personal-access-tokens/new
  2. Token name: Erbario lookup (o come vuoi)
  3. Expiration: 90 giorni / 1 anno / custom
  4. Repository access: Only select repositories → scegli erbario
  5. Repository permissions → Contents → Read and write
  6. Clicca Generate token in fondo
  7. Copia il token (inizia con github_pat_...)
Il token resta nel localStorage del tuo browser. Non passa da nessun server di terze parti. Se un giorno vuoi revocarlo basta cancellarlo da GitHub.
